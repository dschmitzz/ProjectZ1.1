Longest Common Subsequence

Time complexity: O(mn), where m and n are the lengths of the two strings we are getting the LCS of

Algorithm:


Create an n + 1 by m + 1 array (L)
Fill row 0 and column 0 with 0s
For each cell L[i][j], where i <-- 1, n and j <-- 1, m:
    If string1 at i equals string2 at j
    L[i][j] <-- 1 + L[i - 1][j - 1] (above-left diagonal value)
L[i][j] <-- max(L[i - 1][j], L[i][j - 1]) (above value, left value)
Or if above and left values are the same, then take some from the left
