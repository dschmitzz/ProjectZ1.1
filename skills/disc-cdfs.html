Discrete Cumulative Disctribution Function

Given a discrete random variable \(X\), the cumulative distribution function \(F(x)\) satisfies the following properties:

  1. \(F(x) = P(X \leq x) = \sum_{x_i\leq x}f(x_i) \)
  2. \(0 \leq F(x) \leq 1\)
  3. If \(x \leq y\), then \(F(x) \leq F(y)\)
