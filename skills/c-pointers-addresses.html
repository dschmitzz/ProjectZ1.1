Pointers & Addresses

Pointer

A pointer contains a memory address and the type of data stored at that address.

If x is a variable, then *x is a pointer to its address.

Pointers to pointers can be written with **x, and so on. Pointers beyond ***x are very rare.

Pointer Example

char b = 98; // c stores the value 97 or 'b' in ASCII

Void Pointer

Void pointers contain only a memory address, not the type of data stored at the address.
