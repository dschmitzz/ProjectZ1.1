Greatest Common Denominator

The greatest common denominator of integers \(a\) and \(b\) is the largest positive integer \(c\) such that \(c|a\) and \(c|b\).

Euclid's Algorithm

gcd(int a, int b) {
    if a < b, swap(a, b)
    while n != 0 {
        int r = m % n
        m = n
        n = r 
    }
    return m;
}
