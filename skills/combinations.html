Combination

Given a collection of \(n\) distinct objects, an unordered selection of \(r\) objects is called an \(r\)-combination.

number of \(r\)-combinations of \(n\), \(_nC_r = C(n,r) = \left( \begin{array}{c} n \\ r \end{array} \right) = {n! \over r!(n-r)!}\)

for \(0 \leq r \leq n\)
