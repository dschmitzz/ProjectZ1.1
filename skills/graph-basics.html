Graph Basics

Graph

A graph \(G\) is a pair \((V, E)\) where

  • \(V\) is the set of vertices
  • \(E\) is the set of edges (connections between two vertices)

Terminology

We say that an edge is incident with both of the vertices it connects.

Two vertices connected by an edge are adjacent to each other.

A loop edge is an edge that connects a vertex to itself.

A loop-free graph is a graph with no loop edges.

Path, Polygon

\(P_n\): path with \(n\) edges \((n \geq 1)\). has \(n + 1\) vertices
