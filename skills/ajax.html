Ajax

Ajax (Asynchronous JavaScript and XML) is a set of techniques to send and retrieve data between server and client asynchronously and thereby avoid waiting time while the request is being processed.

XMLHttpRequest

The readystatechange handler was the only handler defined in the first version of XMLHttpRequest. Load, error, abort, and timeout have replaced it in later versions to increase clarity and specificity.
