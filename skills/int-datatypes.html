Integer Data Types

Unsigned Integers

Unsigned integers only have a magnitude; they are neither negative nor positive.

Signed Magnitude Integers

Signed magnitude integers are the same as unsigned integers, except the leftmost bit is reserved for the sign.

If the leftmost bit is 0, the number is positive; if the leftmost bit is 1, the number is negative.

1's Complement Integers

A positive integer's 1's complement negative is found by flipping all the bits of its binary form.

2's Complement Integers

A positive integer's 2's complement negative is found by flipping all the bits of its binary form and adding 1.
