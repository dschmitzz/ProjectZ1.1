Permutations

Permutation

Given a collection of \(n\) distinct objects, an ordered arrangement of these objects is called a permutation of the collection.

n permute r, \(_nP_r = P(n,r) = {n! \over (n-r)!}\)

"Permutation" with repetition

Given a set \(S\) within which repetition is allowed, an ordered arrangement of \(r\) elements is called an \(r\)-tuple. Suppose set \(S\) consists of \(n\) elements:

number of \(r\)-tuples = \(n^r\)
