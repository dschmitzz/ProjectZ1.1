Random Variables

Random Variables

In a random experiment, a random variable \(X\) assigns a real number to each possible outcome \(x\) in the sample space.

Discrete Random Variables

A discrete random variable has a finite or countably infinite range.

Examples

number of scratches on a surface, number of phone lines

Continuous Random Variables

A continuous random variable extends over a range (finite or infinite) of real numbers.
