Data Types

Basic Data Types

A char stores 1 byte (8 bits).

A short typically stores 2 bytes (16 bits).

An int typically stores 4 bytes (32 bits).

A boolean (true or false) can be any size an int can be.

Sizes of shorts and ints will vary among different computer architectures. The function sizeof() will return the correct size of any data type in a specific architecture, in bytes.
