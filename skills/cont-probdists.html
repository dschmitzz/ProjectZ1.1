Complete Graphs

Given a set of \(n\) vertices \(V\),

The complete graph on \(V\), \(K_n\), is a loop-free undirected graph where for all \(a,b \in V, a \neq b\), there is an edge \(\{a,b\}\).

The number of edges will be \(\begin{pmatrix} n \\ 2 \end{pmatrix}\).

Example

To make a complete graph from \(V= \{\)\(\}\), the set of edges must be \(E = \) \(\{(a, b), (a, c), (b, c)\}\).

You'll have \(\begin{pmatrix} 3 \\ 2 \end{pmatrix} = {3! \over 2!(3-2)!} = 3\) edges since the graph is composed of 3 vertices.

Complements

Given \(G\), a loop-free undirected graph with \(n\) vertices,

\(G\)'s complement is \(\bar G\)
