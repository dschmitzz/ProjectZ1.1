Continuous Probability Distributions

Probability Density Functions

Given a continuous random variable \(X\), the probability mass function \(f(x)\) satisfies:

  1. \(f(x) \geq 0\)
  2. \(\int_{-\infty}^\infty{f(x)}dx = 1\)
  3. \(P(a \leq X \leq b) = \int_a^b{f(x)}dx = \) area under \(f(x)\) from \(a\) to \(b\) for any \(a, b\)

Inclusive/Exclusive Boundaries are Irrelevant

Given a continuous random variable \(X\), for any \(x_1\) and \(x_2\), $$P(x_1 \leq X \leq x_2) = P(x_1 \lt X \leq x_2) = P(x_1 \leq X \lt x_2) = P(x_1 \lt X \lt x_2) $$
